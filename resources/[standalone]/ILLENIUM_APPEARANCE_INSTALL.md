# illenium-appearance Installation for Qbox

## ⚠️ REQUIRED: Download illenium-appearance

**bub-multichar requires illenium-appearance to function properly.**

### Download Instructions:

1. **Go to the Qbox release:**
   ```
   https://github.com/iLLeniumStudios/illenium-appearance/releases/tag/qbox
   ```

2. **Download the release ZIP file**

3. **Extract to:**
   ```
   resources/[standalone]/illenium-appearance/
   ```

4. **IMPORTANT:** The folder name MUST be exactly:
   ```
   illenium-appearance
   ```
   (Not `illenium-appearance-qbox` or any other variation)

### After Installation:

1. The illenium-qbx-bridge will automatically integrate it with QBX Core
2. Restart your server
3. Verify it loads without errors

### Verification:

After installation, you should see in console:
```
[illenium-qbx-bridge] Server bridge initialized successfully!
[illenium-appearance] Using QBX Bridge for QBX Core compatibility
```
